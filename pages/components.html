Major Components of a Computer

Important components include CPU, RAM, Hard Drive, Motherboard, Power Supply, and Input/Output devices.

Central Processing Unit (CPU)

The CPU is the brain of the computer, responsible for executing instructions and processing data.

Random Access Memory (RAM)

RAM is a type of computer memory that can be accessed randomly. It is used to store working data and machine code.

Hard Drive

The hard drive is used for storing data permanently, including the operating system, applications, and files.

Motherboard

The motherboard is the main circuit board that connects all components of the computer.
